== Description ==

This plugin implements a payment gateway for WooCommerce to let buyers pay with Bitcoin, Ethereum, USDT, and other cryptocurrencies via UniPayment.io.


https://www.unipayment.io

== Installation ==

1. Upload the `unipayment-payment-gateway-for-woocommerce` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. In the WooCommerce Settings page go to the Payment Gateways tab, then click UniPayment.io.
1. Check "Enable UniPayment.io" and enter your UniPayment.io App Id and Api Key.
1. Click "Save changes" and the gateway will be active.
